# Sales Performance & Business Insights Dashboard

## Overview

This project presents an interactive Power BI dashboard developed to analyze sales performance, profitability, target achievement, returns, and delivery efficiency. The dashboard provides business stakeholders with actionable insights through KPI tracking and data visualization.

## Business Objective

The objective of this dashboard is to help management:

- Monitor sales and profit performance
- Track target achievement
- Analyze category and regional performance
- Monitor product returns
- Evaluate delivery efficiency
- Support data-driven decision-making

## Dataset Used

The dashboard was built using the following datasets:

- Orders
- Returns
- People
- Target

## Key Performance Indicators (KPIs)

- Total Sales
- Total Profit
- Total Quantity Sold
- Sales Target
- Sales Variance
- Achievement Percentage
- Returned Orders
- Average Delivery Days
- Delivery Status

## Dashboard Features

### Sales Analysis
- Sales by Category
- Sales by Region
- Sales vs Target Comparison

### Profit Analysis
- Profit by Category
- Profit Performance Monitoring

### Operational Analysis
- Returns Tracking
- Delivery Performance Monitoring

### Interactive Filtering
- Year Filter
- Region Filter
- Category Filter

## Tools & Technologies

- Power BI
- DAX
- Power Query
- Data Modeling
- Data Visualization

## Key Insights

- Achievement Percentage exceeded 100%, indicating target attainment.
- Technology emerged as the highest-performing category.
- Central region generated the highest sales contribution.
- Average delivery performance remained within acceptable limits.
- Returns analysis helped identify potential operational concerns.

## Business Value

This dashboard enables managers to:

- Monitor business performance in real time
- Compare actual performance against targets
- Identify profitable categories and regions
- Improve delivery efficiency
- Make informed business decisions
